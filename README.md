# emoapp

This project is meant to be a tool for tracking emotions, based on topics specified.
The main purpouse of this project is to be able to track emotions and emotional shifts based on topics
At the same time it should be a tool for tracking progress, patterns of thoughts.

For this, I made a distinction for entering different types of journal entries:
- journal entry (default, based on today)
- retrospective (mainly focused on reflections to past events)
- perspective (a glimpse on an emotion in the future.)

Every journal entry can have single value emotional value.

The main purpouse for this is to track via a little "heatmap" of what is/was felt or should be felt during a day
and share this information with others.

## Why? 

Out of curiosity and as a tool for recognizing my own thought / emotional patterns during the day/week/months 
For now, only the current year is supported. I'll change it later on
